# 🛠️ Скрипты автоматизации Git

Набор полезных bash-скриптов для ускорения работы с репозиторием в Ubuntu.

---

## 📂 Интерактивный менеджер веток (`autocheckout.sh`)

Скрипт выводит визуальное графическое меню со списком всех локальных веток. Активная ветка подсвечивается динамически. Вы можете мгновенно переключиться на нужную ветку, просто введя её порядковый номер.

### 🚀 Запуск менеджера

Откройте терминал в папке проекта и выполните одну команду:
```bash
./autocheckout.sh
```

### 📋 Демонстрация интерфейса в терминале:
```text
┌────────────────────────────────────────┐
│      МЕНЕДЖЕР ВЕТОК РЕПОЗИТОРИЯ        │
├────────────────────────────────────────┤
│  ➔  01) main                      (активна) │
│     02) dev                            │
│     03) dev-4                          │
└────────────────────────────────────────┘
🔢 Введите номер ветки для перехода: _
```

> **Важно:** Перед первым запуском любого скрипта не забудьте выдать ему права в Ubuntu: `chmod +x имя_скрипта.sh`.
